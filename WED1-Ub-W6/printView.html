Druckansicht: Hinweise

  • Stellen Sie die Papiergrösse auf A4 Hochformat ein.
  • Stellen Sie die Seitenränder auf 1.5cm, links auf 2.5cm
  • Aktivieren Sie im Druckdialog das Drucken von Hintergrundfarben und Bilder.
  • Aktivieren Sie die Option "Skalierung ignorieren und auf Seitengrösse anpassen"
  • Verwenden Sie Firefox oder Internet Explorer der neusten Generation fürs Drucken.
